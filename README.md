# HikiLibrary AppManagement

## Functions

* Book, Detail Book Management.
* User Management.
* Staff Management.
* Bill Management.
* Borrow Book, Borrower Management.
* Time Table Staff.
* Statistic.

## Technologies

* **Programming Language:** C#
* **GUI Framework:** WPF
